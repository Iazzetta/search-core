# search-core
API de busca (search.com.br) em com Node e Typescript. REST + Graphql e tests com Jest.

### Install & Run
```
docker-compose up --build
```
### Tests
```
npm run test
```
### API
```
    (GET) http://localhost:5000/search
    (POST) http://localhost:5000/add

    (GraphQL) http://localhost:5000/graphql
```
